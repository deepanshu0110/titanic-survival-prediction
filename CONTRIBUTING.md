# Contributing to Titanic Survival Prediction

Thank you for your interest in contributing!

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/<your-username>/titanic-survival-prediction.git`
3. **Create a branch**: `git checkout -b feature/your-feature-name`
4. **Make your changes** and add tests where applicable
5. **Commit**: `git commit -m "feat: describe your change"`
6. **Push**: `git push origin feature/your-feature-name`
7. **Open a Pull Request** against `main`

## Development Setup

```bash
git clone https://github.com/deepanshu0110/titanic-survival-prediction.git
cd titanic-survival-prediction
python -m venv .venv && source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Code Style

- Follow [PEP 8](https://pep8.org/)
- Use descriptive variable names
- Add docstrings to all functions
- Keep functions small and single-purpose

## Reporting Issues

Open a GitHub Issue with:
- A clear title and description
- Steps to reproduce
- Expected vs actual behaviour
- Your Python version and OS
